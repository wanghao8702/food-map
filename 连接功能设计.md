# 「风味连接」功能设计

> 日期 2026-06-04 ｜ 状态：已通过头脑风暴评审，待实现
> 目标：展示《风味人间》每一集内部「食物之间的连接」（同一集把跨地域的食物按主题串联，如面食西传、火腿东西呼应、豆腐外传）。

## 1. 入口与结构

- 顶栏增加视图切换：`风味地图 | 风味连接`（两个标签，互斥）。
- 进入「风味连接」后，下方一排**集标签**：`第1集 山海之间 | 第2集 落地生根 | …`，默认选中第 1 集。
- 选中某集 → 在**世界地图（world.svg）**上只显示该集的菜，并在相连的菜之间画弧线。

## 2. 布局：地理总览

- 复用世界底图与暖色样式。该集的每道菜按真实产地铺在世界图上（中国菜也直接铺在图上，不收成气泡），用其美食 SVG 小图标（复用现有 pin 外观）。
- **中国菜在世界图上的定位**：world.svg 为 Miller 投影，无法用经纬度线性。采用「China 国家 path 的包围盒 + 中国地理范围线性映射」：把菜的经纬度在中国范围（lng 73.55–134.78 / lat 18.16–53.56，取自 china.svg 的 geoViewBox）线性映射到 world.svg 中 `id="CN"` path 的 bbox 内。海外菜沿用「国家 path 几何中心」。近似但足够（连接图重在关系而非精确省份）。
- 同位置防重叠复用 `pixelSpread`。

## 3. 连接弧线

- 每条连接画一条二次贝塞尔曲线（控制点在两点连线中垂方向偏移，形成弧）。
- **先用颜色区分主题**，不常显文字；颜色取自一组暖色可辨色板，按主题分配。
- 交互：
  - **hover/点击弧线** → 该弧（及同主题整条风味链）高亮加粗，并显示主题文字标签（如「面食西传」）。
  - **点击节点（菜）** → 弹出原有右侧详情面板。
- 支持拖动 / 滚轮缩放 / 双击复位（复用现有 viewBox 平移缩放）。

## 4. 数据模型

- 连接是**集内**关系，存在每集 `episodes/fwX-YY.json` 里，新增 `links` 数组：
  ```json
  "links": [
    {"a": "fw1-01-mizhi-huofang", "b": "fw1-01-iberico", "theme": "腌制风干"},
    {"a": "fw1-01-yangyu-jiaotuan", "b": "fw1-01-aligot", "theme": "土豆"}
  ]
  ```
- `a`/`b` 为 dish id（同集内）；`theme` 为连接主题文字（hover 时显示，并用于配色分组）。
- 主题→颜色映射在 `src/connect.js` 内维护（一组暖色）。

## 5. 代码结构

- `src/connect.js`（新增）：`renderConnections(episodeId)` —— setBase(WORLD)、按该集 links 涉及的菜用 worldPixel 放点、画弧、绑定交互；导出供 index.html 的视图切换调用。
- `index.html`：顶栏加视图切换 + 集标签容器；按钮逻辑在内联模块里接 `map.js` 与 `connect.js`。
- `src/map.js`：暴露必要的工具（`project`/`regionCenter`/`pixelSpread`/平移缩放）或将其抽到可复用处；连接视图与地图视图互斥渲染到同一个 `#map`。
- `styles.css`：视图切换、集标签、弧线样式。

## 6. 各集连接（首批，取纪录片明确点出的关系）

- **第1集**：火方↔伊比利亚火腿（腌制风干）；洋芋搅团↔阿力高（土豆）；太湖蟹↔荷兰蟹（大闸蟹）；太湖蟹↔秃黄油（蟹·江南）。
- **第2集**：桑嘎↔石子馍、石子馍↔枕头馍（面食西传）；Chifa云吞↔木薯五花（秘鲁中餐）；马介休球↔非洲鸡↔Tacho（澳门土生葡菜）。
- **第3集**：白族生皮↔罗布烤鱼↔叫花鸡（火与烤）；蟹壳黄↔锅魁（烧饼演化）；黄豆腐↔麻婆豆腐、黄豆腐↔绢豆腐、麻婆豆腐↔绢豆腐（豆腐 / 豆腐外传）。
- 第4/5/6 集：随数据录入时一并补充。

## 7. 验收

1. 顶栏可在「风味地图 / 风味连接」间切换；连接视图有集标签、默认第1集。
2. 选中某集，世界图上只显示该集的菜（含中国菜按地理铺开），相连菜之间有彩色弧线。
3. hover/点击弧线 → 高亮 + 显示主题文字；点击节点 → 详情面板。
4. 跨境连接（如桑嘎↔石子馍、黄豆腐↔绢豆腐）的弧线横跨欧亚 / 太平洋，清晰可见。
5. 拖动/缩放/双击复位可用；控制台无报错。
