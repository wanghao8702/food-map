# 美食 SVG 视觉规范（扁平填色）
- 画布：viewBox="0 0 120 120"，主体居中，四周留白约 12。
- 描边：统一深棕 #7a2a1c，stroke-width 1.6，stroke-linejoin/linecap=round。
- 调色板（暖色）：主红 #e8623f / 亮橙 #f0a05a / 黄 #f4c95d / 绿 #7fa860 / 米白 #fbf3e0 / 棕 #b9763c。
- 风格：实色填充 + 同色系深色描边；高光用 1–2 个白色小圆/弧。无渐变、无阴影、无外部字体。
- 容器元件：碗(ellipse+口沿弧)、盘(扁椭圆)、砧板、串签；从这些基础形状组合具体菜品。
- 每个文件是自包含 `<svg>`，不引用外部资源。
